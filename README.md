# TypingShooter

迫りくる敵に**ロックオン**し、頭上に表示された**お題をタイピング**して撃ちまくる、単一HTMLのタイピングシューティングゲームです。プレイヤーはサブマシンガンを構えたランボー風キャラクター。1打ごとに「ダンダンダン!」と発砲し、連打ヒートで音とエフェクトが3段階にエスカレートします。全10ステージ＋無限に続くエンドレスモード、雑魚全滅後に登場するボス戦を収録。日本語（ローマ字入力・複数表記対応の `shi/si`・`cha/tya`・促音・撥音）と英語の切り替えに対応しています。

ビルド不要・外部素材/外部通信なしの単一 HTML ファイル（`TypingShooter.html`）で完結し、Google / YouTube などの Playable 配信を想定して設計されています。描画は Canvas 2D、効果音は Web Audio API で全て合成しており、画像・音声アセットを一切持ちません。

## 遊び方

### PC（キーボード）
- `D` / `F` … マーカーを移動（D=まえ / F=つぎ。Tab でも次へ）。※未ロック時のみ
- `Space` … マーカー位置の敵にロックオン ⇄ 解除（トグル）
- 文字キー … **ロックオン中のみ**タイピング判定。お題を打ち切ると撃破
- マウスクリック … 敵をクリックでロックオン
- `Esc` … タイトルへ戻る

マーカー＝白の点線リング、ロックオン＝赤の実線リングで表示されます。

### スマホ（タッチ）
- 敵を**タップ**してロックオン、画面キーボードで**入力**
- 未ロック状態でタイプすると、入力文字に一致する敵を近い順で自動ターゲット

## ローカルでの起動

`TypingShooter.html` をブラウザで開くだけです（ローカルサーバー不要）。

## テスト

Playwright（Chromium）による自動テストを同梱しています。

```bash
npm install
npx playwright install chromium   # 初回のみ
npm test                          # node test.js && node test5.js
```

`test.js` はローマ字エンジン・キー操作・ステージクリア・コンソールエラー0を、`test5.js` は全日本語ワードの完走と隊列システムを検証します（exit 0 = 合格）。`test2.js`〜`test4.js` も同梱しており、個別に `node test2.js` 等で実行できます。
